**Your Squarespace CSS sidekick. Powered by a decade of teaching, hosted here, updated all the time.**

Custom Codey is an AI-powered skill for Claude that helps you customize your Squarespace website with vanilla CSS and basic JavaScript. Ask for the change you want, get working code, ship it.

## What makes Codey different

Most CSS tools know *what* a selector is. Codey knows *why* yours isn't working.

- **Built on 10 years of Squarespace teaching.** Every entry in this repo comes from real student questions, not scraped from docs.
- **Knows the quirks.** The "underline" under your active nav link isn't an underline. `:active` doesn't mean "current page." Your mobile menu button uses a different selector when it's open. Codey knows.
- **Version-aware.** Works across Squarespace 7 and 7.1, including theme-specific selectors for Brine, Bedford, Pacific, and York.
- **Always current.** Squarespace changes things. This repo updates often. Ask Codey to "check for updates" any time.
- **Beginner-friendly.** No jargon. Copy, paste, save, refresh — Codey walks you through it.
