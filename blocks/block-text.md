## About

Express yourself and share your ideas with **Squarespace text blocks**.

The Squarespace text block is a versatile tool that lets you write anything from short blurbs to lengthy blog posts.

You can easily format your text with standard text formatting options like bold, italics, headings, and bullet points to make your content clear and engaging.

There are additional style features like text highlight, color options, and more!

## Snippets

Using any of the code names above, you can get really creative with your text! Here are some examples of changes you can make with CSS, using the code name for a Heading 1 text type.

- Changing text color
    
    ```css
    H1 {color: purple}
    ```
    
- Creating two columns of text
    
    ```css
    H1 {column-count: 2; column-gap: 3rem;}
    ```
    
- Making text uppercase
    
    ```css
    H1 {text-transform:uppercase}
    ```
    

## Pre-made Code Styles

Here are some premade code snippets for creative text that you can try!

These examples use an h1 text; change that code name to the type of text you want to use.

---

**Outlined Text**

![Outline.png](Text%20Block/Outline.png)

```css
h1 {color: transparent!important;
 -webkit-text-stroke-width: 2px;
 -webkit-text-stroke-color: #000000;
}
```

---

**Image Overlay Text**

![Image Overlay.png](Text%20Block/Image_Overlay.png)

```css
h1  { color: #FFFFFF;
 -webkit-text-fill-color: transparent; 
 background: -webkit-linear-gradient(transparent, transparent),
 url (your-image-url-here) repeat; 
 background: -o-linear-gradient(transparent, transparent);
 -webkit-background-clip: text;
}
```

---

**Vertically Aligned Text**

Displays on table and desktop only

![Vertical .png](Text%20Block/Vertical_.png)

```css
@media screen and (min-width: 640px) { 
 h1 { writing-mode: vertical-rl; transform: rotate(180deg); 
 margin: 0 auto; 
 position: relative; 
 right: 10px; 
} 
}
```

---

**Low Highlight Text**
This will add a background color to the bottom half of a text block.

```css
h1 {background: 
 linear-gradient(to bottom, #FFF 50%, #50bdb8 50%)
}
```

---

**Style selected text**
Any text that is highlighted by a cursor can have a custom style too!

```css
::selection {color:#FFFFFF; 
 background: #50bdb8;
}
```

## CSS Properties for Text

This is a list of common properties you can use to change the style of text blocks in Squarespace.

| Property | What It Does | H1 Example |
| --- | --- | --- |
| color | Changes the color of the font | h1 {color: purple} |
| background-color | Gives the text a background color | h1 {background-color: red} |
| font-size | Changes the font size | h1 {font-size: 30px} |
| font-weight | Changes the font thickness; you can use numbers (100-900) or text (thin, normal, bold, bolder) but this will only work if your font supports multiple weights! | h1 {font-weight: bolder} |
| text-transform | Changes capitalization | h1 {text-transform:uppercase} |
| letter-spacing | Adjusts the space between characters | h1 {letter-spacing: 2px} |
| line-height | Adjusts the space between lines of text | h1 {line-height: 20px |

**Multiple text properties in one line**

You can use CSS to apply a property to part of a line of text using the **bold** (strong) or *italic* (em) style options. 
This example will change the font family for any bold h1 text, and give a background color only to the italic text:

```css
h1 {font-family: decorative} 
h1 em{background-color: pink}h1 strong{font-family: serif!important}
```
