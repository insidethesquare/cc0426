# Map Block

Type: Content Block
Compatible Versions: 7, 7.1
Last Updated: April 23, 2026

## About

Show a physical location for a business or event on your website using a **Squarespace map block.**

Map blocks display a visual map for a specific address using the Google map data.

When you place a location pin, website visitors can click the pin to open the map in Google Maps for easy navigation. The map will resize to fit any screen, so it looks great on phones, tablets, and computers.

**Important info to know:**

You can only list one location per map block.

Squarespace can't display business information on the map itself, but you can provide a link to a third-party tool like your Google Business Profile.

**What you can customize about the map block using Squarespace**

The only design features we have for the map block at this time are found on the **design** tab, you can change the map color to a different premade **color** setting (Original, Grayscale, Satellite, Dark, Light, Blue). This is also where you can choose to display or hide the map **labels**, **terrain**, and **zoom** **controls**.
