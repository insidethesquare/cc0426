# Markdown Block

Type: Content Block
Compatible Versions: 7, 7.1
Last Updated: April 23, 2026

## About

Use markdown language to add content to your site with a **Squarespace markdown block.**

Markdown blocks are a convenient way to add website content using basic HTML, hypertext markup language.

For advanced website developers who are more comfortable with that code language, this content block is a great way to customize Squarespace without using standard content block formatting.

**Pro tip:** a well known Squarespace bug, your cursor might not align within the content inside a markdown block while editing.

**What you can customize about the markdown block using Squarespace**

There are no design features for the markdown block at this time. Text content inside a markdown block will be formatted to match the text style assigned in your **site** **style** menu.

## Selectors

| Entire Markdown Block | **.markdown-block** |
| --- | --- |
| Heading 1 | # Heading 1 |
| Heading 2 | ## Heading 2 |
| Heading 3 | ### Heading 3 |
| Heading 4 | #### Heading 4 |
| Heading 5 | ##### Heading 5 |
| Bold | **bold** |
| Italic | *italic* |
| Superscript | normal<sup>superscript</sup> |
| Subscript | normal<sub>subscript</sub> |
| Block Quote | > Quote text |
| List Item | * item one* item two |
