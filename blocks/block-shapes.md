# Shape Block

Type: Content Block
Compatible Versions: 7.1
Last Updated: April 23, 2026

## About

Get creative with your design using customizable **Squarespace shape blocks!**

Squarespace shape blocks are like your website's colorful building blocks. They let you add fun geometric shapes (think squares, circles, triangles, and more!) in all sorts of sizes and colors.

These shapes are a fantastic way to add a decorative touch, break up text, or even highlight specific areas of your content.

They're super versatile; you can customize them to perfectly match your website's style and personality!

Shape blocks have one main selector: **.sqs-shape**

## Snippets

- Simple drop shadow for a shape block
    
    You can add a drop shadow to a shape block using the design menu, but if you want to apply a shadow to all shape blocks on your site, this code is a fast way to do it!
    
    ```css
    .sqs-shape{
    filter:drop-shadow(5px 5px 5px #333)
    }
    ```
